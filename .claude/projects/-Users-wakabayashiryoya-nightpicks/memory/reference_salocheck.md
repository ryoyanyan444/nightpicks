---
name: SaloCheckからの送客
description: SaloCheck（セットサロンSaaS）からNIGHTPICKSへのトラフィック流入がある
type: reference
---

SaloCheck（salocheck.com）はユーザーが運営するセットサロン向けSaaSサービス。
NIGHTPICKSへバナー広告経由で送客している（utm_source=salocheck&wid={worker_id}）。
既にトラフィック獲得ができている状態。

**How to apply:** SaloCheckからの流入を前提にSEO・マネタイズ戦略を考える。クロスドメインの送客はSEOに直接効果はないが、間接的にユーザー行動指標を改善する。
